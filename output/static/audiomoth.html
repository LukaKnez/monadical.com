Get demo firmware working on audiomoth 2. Get access to last sample (+ last 10 samples?) in main record loop 3. write `does_contain_gunshot(samples) -> true/false` 4. flash LED red quickly if it returns `true` 5. get it working while recording to SD card at high sample rate and measure power use ### Phase 2: Get the LoRa mesh communication working 1. Get LoRa mesh or P2P connections working between gateway and leaf node 2. listen for GPIO high input on leaf node, send message to gateway 3. gateway sends SMS/push notification to user's phone saying "Gunshot detected" ### Phase 3: putting it all together 1. Connect GPIO "gunshot detected" pin to LoRA GPIO 2. Send ping on gunshot detection via LoRA, while simultaneously recording to SD card 3. ambitious: add ability to send audio over LoRA/LTE/Wifi/Bluetooth to nearby ranger device without needing to remove microSD card
